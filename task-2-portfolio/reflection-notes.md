## Task 2 Write-up

For Task 2, I focused on building a basic professional digital presence by creating profiles on GitHub, LinkedIn, and HackerRank. In today’s time, having an online presence is very important for students as it helps in learning, networking, and career opportunities.

GitHub is a platform used for storing code and managing projects. I plan to use GitHub as my main portfolio where I can upload my coding projects and track my progress over time. It will help me showcase my skills and build real-world projects during my B.Tech journey.

LinkedIn is a professional networking platform. I have added my educational details to start building my profile. In the future, I will use LinkedIn to connect with seniors, learn from professionals, and explore internship opportunities.

HackerRank is a coding practice platform that helps improve problem-solving skills. I plan to use it regularly to practice coding problems and strengthen my programming knowledge. It will also help me prepare for technical interviews.

Overall, this task helped me understand the importance of building a professional digital identity from the beginning.
