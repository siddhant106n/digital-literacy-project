# Student Digital Portfolio

Welcome to my digital portfolio setup for the CSE0001 Digital Literacy course. In this task, I created and updated my professional profiles on important platforms to start building my online presence.

## 1. GitHub

**Platform Purpose:**  
GitHub is a platform used for storing code, managing projects, and collaborating with other developers.

**My Strategy:**  
I will use GitHub as my main portfolio to upload my projects and coding work. Over the next few years, I plan to share my programming projects and improve my skills by working on real-world applications.

## 2. LinkedIn

**My Strategy:**  
I have created my profile with my education details. I plan to use LinkedIn to connect with seniors and professionals, learn about opportunities, and apply for internships in the future.

## 3. HackerRank
