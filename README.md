## Project Overview

This repository contains my Digital Literacy Portfolio, created as the final deliverable for the CSE0001 course. As a student, this project helped me understand and apply essential digital skills required in today’s world.

The main objective of this project is to build awareness about digital tools, online communication, and cybersecurity practices. It also focuses on creating a professional online presence and using digital platforms responsibly.

All tasks and activities are organized systematically in this repository.

---
